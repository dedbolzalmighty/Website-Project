Enrollment Procedure

Gamit ang ano man sa sumusunod:

  • Deskyop

  • Laptop

Gamit ang ano man sa sumusunod:

Sa inyong web browser, i-type ang IP Address na 110.93.74.76

Sagutan ang mga sumusunod:

  • LRN (LEARNER’S REFERENCE NUMBER)

  • PERSONAL INFORMATION

  • EDUCATIONAL BACKGROUND

  • SOCIO-ECONOMIC

  • URS CAMPUS APPLIED FORN

E-MAIL VALIDATION

  • Make sure that your information is complete.

  • Go to your e-mail account and find the e-mail verification code in your inbox.

  • Copy the verification code and paste it.

  • If you didn’t receive an e-mail from the system, then click Resend Code and go back to your e-mail account.

  • You will know that your account is fully validated if you received this message:
    “Your e-mail has been validated successfully.”

UPLOADING OF DOCUMENTS

  • The last step in order to be validated is to upload the required documents.

  • Make sure that the file size is 2MB only.

The system only accepts PDF file.

Application Done

  • Maghintay na lamang ng e-mail kapag nabasa mo ito ‘Application Done’ o maghintay ng announcement mula sa coordinator ng inyong campus.

Required Documents for FRESHMEN:

  • Form 138 (General Average in Grade 12 First Semester)

  • Certificate of Good Moral Character

  • PSA Birth Certificate

  • 2 Copies of 2x2 recent ID picture (white background)

NOTE: Please wait for further announcement regarding the online application process.

Required Documents for TRANSFEREES:

  • Transfer Credentials (Honorable Dismissal)

  • Transcript of Records / Scholastic Record with General Weighted Average (for evaluation purposes)

  • Certificate of Good Moral Character

  • PSA Birth Certificate

  • 2 Copies of 2x2 recent ID picture (white background)

Munting Paalala:

  • Do not leave any empty fields. All fields are required if necessary.

  • Please be careful. You won’t be able to change the date once this form is submitted.

  • You will be given an ‘Application ID’ after completing this form.
